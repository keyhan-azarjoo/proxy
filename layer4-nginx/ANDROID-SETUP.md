## تنظیمات مورد نیاز

در اپلیکیشن، این اطلاعات را وارد کنید:

- **Host**: IP سرور
- **Port**: **443** (مهم!)
- **Username**: نام کاربری
- **Password**: رمز عبور

---

## نکات مهم

- حتماً پورت **443** را استفاده کنید (لایه ۴)
- این پورت شبیه HTTPS است و فیلتر شدن آن سخت‌تر است
- نام کاربری و رمز عبور را که در سرور ساختید وارد کنید

✅ اتصال برقرار شد!

## Required Settings

In the app, enter this information:

- **Host**: Server IP
- **Port**: **443** (important!)
- **Username**: Your username
- **Password**: Your password

---

## Important Notes

- Make sure to use port **443** (Layer 4)
- This port looks like HTTPS and is harder to filter
- Enter the username and password you created on the server
